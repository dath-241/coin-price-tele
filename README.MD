## Hướng dẫn cài đặt
Sử dụng .env.template và thay thế bằng token tele của bạn

Tải xuống các package: ```go mod tidy```

Để khởi động bot, cd vào thư mục src, chạy ```go run main.go```

Để khởi động mock be server, cd vào thư mục mock-be
```
npm install
npm run start:dev
```
